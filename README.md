# Gold Price Prediction using Machine Learning

## Project Overview

This project predicts gold prices using Machine Learning techniques.

The model analyzes economic indicators such as stock market indices, silver prices, oil prices, and currency exchange rates to estimate gold prices.

---

## Features

- Data Cleaning
- Correlation Analysis
- Heatmap Visualization
- Distribution Analysis
- Random Forest Regressor
- Model Evaluation using R² Score
- Gold Price Prediction System

---

## Dataset Features

- Date
- SPX
- USO
- SLV
- EUR/USD

Target Variable:

- GLD (Gold Price)

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## Project Workflow

1. Import Libraries
2. Load Dataset
3. Explore Dataset
4. Correlation Analysis
5. Visualize Data
6. Select Features and Labels
7. Split Dataset
8. Train Random Forest Regressor
9. Evaluate Model Performance
10. Predict Gold Prices
